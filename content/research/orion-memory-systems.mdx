## Memory changes the shape of software

Agentic systems become more useful when they can preserve context, preferences and operating history.

The research question is not only what an agent can do in one session, but what it can learn across time.

## Research direction

Orion explores agentic operating systems, memory, GUI control and autonomous product development.

Public research notes describe the direction without exposing private memory schemas, prompts, runbooks or implementation details.
