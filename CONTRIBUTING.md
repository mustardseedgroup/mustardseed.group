# Contributing

This repository is public and contains only the public-facing Mustard Seed Group website.

## Rules

- Do not add private product source code.
- Do not add prompts, runbooks, agent orchestration, memory systems or lead intelligence.
- Do not add client data, secrets or environment variables.
- Keep public writing restrained, accurate and in British English.
- Draft content must go through pull request review before publishing.

## Content

Content lives in:

- `content/blog/`
- `content/research/`
- `content/companies/`
- `content/pages/`
- `content/drafts/`

Use MDX frontmatter and keep public boundaries clear.
